# cnsite-leisu-archive

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库内收录的页面均为静态 HTML 文件，不依赖后端服务，可直接通过浏览器访问。

本仓库不针对任何特定域名或网站，仅作为资料存档与页面展示用途。

## 目录说明

```
├── pages/          # 存放归档的 HTML 页面
├── assets/         # 页面引用的资源文件（CSS、JS、图片等）
├── index.html      # 仓库主页（可选）
└── README.md       # 本文件
```

- `pages/`：所有独立 HTML 页面文件均放置于此目录下，按需可进一步按子目录分类。
- `assets/`：页面所需的公共资源文件，包括样式、脚本、字体、图片等。
- `index.html`：若需要提供一个导航或总览页面，可放置于此。

## 页面归档说明

- 所有归档页面均为完整独立的 HTML 文件，部分页面可能引用 `assets/` 目录下的资源。
- 页面内容可能包含静态文本、图片、表格、链接等常见 HTML 元素。
- 页面不包含任何动态交互或后端逻辑，所有内容在加载时即确定。
- 归档页面仅作资料保存与展示，不保证其内容时效性或准确性。

## 维护说明

- 本仓库由维护者不定期更新，新增或移除页面均会通过 commit 记录。
- 如发现页面显示异常或资源缺失，欢迎提交 Issue 反馈。
- 原则上不接收外部直接提交的页面文件，如有归档需求请先联系维护者。
- 本仓库内容仅作存档用途，不提供任何形式的服务承诺或技术支持。

## 其他

- 所有页面均可在支持 HTML 标准的浏览器中正常打开。
- 仓库使用 Git 进行版本管理，历史记录可供查阅。
